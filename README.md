# 网易云音乐播放列表备份工具

解决网易云音乐 Windows 客户端没有播放列表历史记录功能的痛点。  
当你不小心点了一首歌导致播放列表被替换时，可以轻松恢复之前的列表。

## 功能

- **静默后台运行**: 守护进程通过 `pythonw.exe` 启动，完全无窗口
- **自动跟随启动**: 检测网易云音乐进程，应用启动时自动开始监控，关闭时暂停
- **大幅变化弹窗**: 播放列表疑似被覆盖时自动弹窗，一键恢复
- **自动监控备份**: 播放列表文件变化时自动创建备份
- **手动备份**: 随时手动创建备份快照
- **智能去重**: 内容未变化时自动跳过，不浪费空间
- **查看详情**: 查看任意备份中的完整歌曲列表
- **差异对比**: 对比两个备份或备份与当前列表的差异
- **一键恢复**: 恢复任意历史备份，恢复前自动保护当前列表
- **开机自启动**: 一键安装/卸载开机自启动
- **单实例保护**: 自动防止重复启动
- **最多保留 100 个备份**，自动清理最旧的

## 快速开始

### 推荐用法：静默守护模式

1. 双击 `安装开机自启.bat`，设置开机自动启动并立即运行守护进程
2. 正常使用网易云音乐，一切在后台自动完成
3. 当播放列表被意外覆盖时，会自动弹窗提醒并提供恢复选项
4. 如需手动管理，使用 CLI 命令（见下方）

### 双击快捷操作

| 文件 | 用途 |
|------|------|
| `安装开机自启.bat` | 安装开机自启动并启动守护进程 |
| `卸载开机自启.bat` | 卸载开机自启动并终止守护进程 |
| `启动监控.bat` | 前台模式启动监控（带控制台窗口） |
| `手动备份.bat` | 立即创建一个备份 |
| `查看备份.bat` | 列出所有备份 |

### 命令行

```bash
# ─── 守护进程管理 ───
python install.py install      # 安装开机自启动
python install.py uninstall    # 卸载开机自启动
python install.py status       # 查看状态
python install.py start        # 立即启动守护进程
python install.py stop         # 终止守护进程

# ─── 前台监控（调试用）───
python main.py watch

# ─── 备份管理 ───
python main.py backup             # 手动备份
python main.py backup --force     # 强制备份
python main.py list               # 列出所有备份
python main.py detail 3           # 查看第 3 个备份（前 20 首）
python main.py detail 3 --full    # 查看完整歌曲列表
python main.py compare 2 5        # 对比第 2 和第 5 个备份
python main.py compare 3 --current  # 对比备份与当前列表
python main.py restore 3          # 恢复第 3 个备份
python main.py restore 3 -y       # 跳过确认提示
python main.py delete 5           # 删除备份
```

## 工作原理

```
┌─────────────────────────────────────────────────┐
│  守护进程 (daemon.py, pythonw.exe 静默运行)     │
│                                                 │
│  1. 每 5 秒检测网易云音乐进程                    │
│  2. 检测到启动 → 开始监控 playingList 文件       │
│  3. 文件变化 → 自动备份                         │
│  4. 分析变化幅度:                               │
│     · 重叠率 < 40% 或 歌曲数下降 > 50%          │
│     → 弹窗警告，询问是否恢复                     │
│  5. 网易云关闭 → 停止监控                        │
└─────────────────────────────────────────────────┘
```

## 典型使用场景

### 日常防护（推荐）
安装开机自启即可，完全无感。播放列表被覆盖时会自动弹出警告窗口。

### 播放列表已经被覆盖了
1. `python main.py list` — 找到想恢复的备份
2. `python main.py compare 5 --current` — 确认差异
3. `python main.py restore 5` — 恢复
4. 重启网易云音乐客户端

## 依赖

- Python 3.6+

## 文件说明

```
CloudMusic Playlist/
├── main.py              # CLI 主程序（备份管理命令）
├── daemon.py            # 后台守护进程（静默运行+弹窗）
├── backup_manager.py    # 备份管理核心逻辑
├── watcher.py           # 前台文件监控模块
├── install.py           # 开机自启动安装/卸载
├── 安装开机自启.bat      # 一键安装
├── 卸载开机自启.bat      # 一键卸载
├── 启动监控.bat          # 前台模式启动
├── 手动备份.bat          # 快捷手动备份
├── 查看备份.bat          # 快捷查看备份列表
├── backups/             # 备份文件存放目录（自动创建）
│   ├── backup_meta.json # 备份元数据索引
│   └── playlist_*.json  # 各个备份文件
└── logs/                # 日志目录（自动创建）
    ├── watcher.log      # 前台监控日志
    └── daemon.log       # 守护进程日志
```

## 注意事项

- 恢复备份后需要**重启网易云音乐客户端**才能生效
- 守护进程完全在后台运行，不影响网易云音乐的正常使用
- 守护进程有单实例保护机制，不会重复启动
- 备份文件保存在工具目录下的 `backups/` 文件夹中
- 日志文件位于 `logs/daemon.log`，可用于排查问题
